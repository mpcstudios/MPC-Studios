## Brand Voice & Tone

### Voice Characteristics

- **Confident but not corporate.** We know what we're doing. We don't need jargon to prove it.
- **Direct.** Say what something is, not what it "leverages" or "synergizes."
- **Warm and practical.** South Texas practical — no filler, no fluff.
- **Second person.** Talk to the client: "your business," "your team," "your results."
- **Active voice.** "We create and post content" not "content is created and posted."

### Vocabulary — Use These

| Use | Instead Of |
|-----|-----------|
| Investment | Cost, fee, price |
| Services | Products |
| Strategy calls | Check-ins |
| System | Bundle, package |
| Compounding | Stacking, accumulating |
| Marketing department | Marketing vendor |
| Entry point | Cheap option |
| Accessible | Affordable |

### Vocabulary — Avoid These

| Avoid | Why |
|-------|-----|
| Package, bundle | These are tiers, not bundles. Bundles sound disposable. |
| Cheap, affordable | Use "accessible" or "entry point." We sell value, not discounts. |
| Guarantee | Marketing has no guarantees. We set measurable goals and track against them. |
| Unlimited | Nothing is unlimited. Every tier has clear deliverables. |
| Synergy, leverage, holistic | Corporate buzzwords. Say what you mean in plain English. |

### Tone by Context

- **Homepage / Hero:** Confident, aspirational, outcome-focused. Lead with what the client gets.
- **Service pages:** Factual, specific, deliverable-focused. Tell them exactly what's included.
- **Tier comparison:** Clean, scannable, no persuasion — let the value speak.
- **FAQ:** Conversational, direct, honest. Answer the question first, then explain.
- **Case studies:** Storytelling. Start with the client's problem, show the journey, end with results.
